# I18n

Use this guide when adding or moving user-facing copy, validation messages, mailer text, controller notices, or view labels.

Default locale: **French** (`fr`). English (`en`) mirrors the same key tree.

## File layout

One file per model / surface, under the locale folder:

```text
config/locales/<locale>/<model_or_surface>.yml
```

| File | Owns |
| --- | --- |
| `event.yml` | `events.*` + `activerecord` for `Event` |
| `event_participant.yml` | `event_participants.*` + AR for `EventParticipant` |
| `event_team.yml` | `event_team.*` + AR for `EventTeam` |
| `friendship.yml` | `friendships.*` + AR for `Friendship` |
| `notification.yml` | `notifications.*` |
| `user.yml` | `users.*` + AR for `User` |
| `layout.yml` | `layouts.*`, `pwa.*` |
| `pages.yml` | Landing / marketing (`pages.*`) |
| `admin.yml` | Back-office (`admin.*`) |
| `shared.yml` | Shared `activerecord.errors.messages`, `errors.messages` |
| `devise.yml` | Devise keys (Devise-required paths — do not rename) |
| `date.yml` | Rails `date` / `time` / `datetime` formats |

Do **not** create `feature.fr.yml` names or `config/locales/models/` folders.

## Key hierarchy (content types)

Inside each feature namespace, group by content type:

```yml
fr:
  events:
    titles:        # page / section headings
    flash:         # controller notices / alerts
    actions:       # buttons, links, submit labels
    confirms:      # turbo_confirm / destructive prompts
    empty:         # empty states
    labels:        # visible labels, badges, status text
    placeholders:  # form placeholders
    aria:          # accessibility labels
    invite:        # feature-specific UI blocks (optional)
  activerecord:
    attributes:
      event: ...
    errors:
      models:
        event: ...
```

Same tree for `en:` with English copy.

## Model errors

Prefer symbol errors in models (`errors.add(:attr, :symbol)`), then define the message in the model’s locale file:

```yml
fr:
  activerecord:
    errors:
      models:
        friendship:
          attributes:
            receiver_id:
              self: "Vous ne pouvez pas vous envoyer une demande d'amitié à vous-même."
          already_exists: "Une demande d'amitié existe déjà entre ces utilisateurs."
```

## Rails usage

```erb
<%= t("events.titles.home") %>
<%= t("friendships.flash.create.success", name: user.first_name) %>
```

```ruby
redirect_to friendships_path, notice: t("friendships.flash.update.success", name: user.first_name)
```

Avoid hard-coded user-facing strings in views, controllers, mailers, and model validation messages.

## Locale switching

Preferred locale is stored in a **cookie** (`locale`), not in the database.

Resolution order per request:

1. Valid `cookies[:locale]` (`fr` or `en`)
2. Browser `Accept-Language` (first matching available locale)
3. `I18n.default_locale` (`fr`)

Switch from **Edit account** (`/users/edit`) via `PATCH /locale/:locale` (`LocalesController`).

See `app/controllers/concerns/set_locale.rb`.

## Devise

Keep Devise’s expected key paths (`devise.sessions.*`, `devise.mailer.*`, etc.). Reorganize only within those Devise namespaces when needed.
